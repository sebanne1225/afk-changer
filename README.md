# AFK Changer

VRChat アバターの AFK アニメーションを非破壊で入れ替える NDMF プラグインです。

> **WIP**: このツールは開発中です。まだ公開・配布の準備はできていません。

## 何ができるか（予定）

- AFK モーション（AnimatorController）をアバターに設定するだけで、ビルド時に Action Layer の AFK ステートを自動で入れ替えます
- 元のアバターには直接変更を加えない非破壊設計です（NDMF）

## 対応環境

- Unity `2022.3`
- VRChat SDK（Avatars）
- NDMF
- VCC / VPM ベースの VRChat プロジェクトを推奨します
